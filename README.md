# Personal Expenses Analysis
Simple Python project to analyze monthly household expenses. 
The goal of this project is to demonstrate basic data analysis skills, clean code structure and result visualization using personal data.
This project loads expense data from a CSV file, performs basic data cleaning, analyzes total and category-based expenses, and visualizes the results.
